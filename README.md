# 👟 Nase's Best Shoes Collection

Welcome to the official digital showroom for *Nase's Best Shoes*. This project is a two-page, high-performance web catalog designed for a seamless mobile and desktop shopping experience.

## ✨ Features
* *Graphical Dark Mode:* A sleek black-themed UI that makes products stand out.
* *Smart Slider:* An automated image showcase on the home page.
* *Full-Screen Focus:* Tap any shoe to see it in high-definition detail.
* *Direct Inquiry:* Integrated WhatsApp API for instant customer support.
